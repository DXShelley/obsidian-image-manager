# 文档索引

此目录采用中英文双版本组织：

- 中文文档默认使用原文件名，例如 `user-guide.md`。
- 英文镜像使用 `.en.md` 后缀，例如 `user-guide.en.md`。
- 新增或更新文档时，应同步维护两个版本，避免内容漂移。
