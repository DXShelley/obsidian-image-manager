# Obsidian Image Manager

A comprehensive image management plugin for Obsidian.

## Current Features

- Paste images into notes and auto-save them into a configurable folder.
- Convert pasted images to the configured default format.
- Generate names with variables such as `{noteName}`, `{fileName}`, `{date}`, `{time}`, and `{random}`.
- Format inserted image links as either Obsidian Wiki links or standard Markdown links.
- Process all images referenced in the current note.
- Use image context menu actions for copy, compress, convert, rotate, flip, and delete.
- Open basic note and folder image galleries.

## Development

```bash
npm install
npm run type-check
npm run test
npm run build
```
